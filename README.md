# vue-element-admin-change  #

## 项目简介

## 在vue-admin-template基础版本上进行了扩展，引入了集成版的选项卡，公共配置。 ##

## 新增@/settings.js文件中的routesRole来切换前端或者后端控制路由 （默认前端控制） ##

## 通过Vue.directive绑定全局按钮级权限指令v-permission(系统设置有所体现) ##

## 人员管理，角色管理 ##
